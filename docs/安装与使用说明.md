# FlowType 安装与使用说明

## 安装

1. 从 release 页面下载 `FlowType Setup 0.2.0.exe`，或在源码目录运行 `npm run dist` 自行构建安装包。
2. Windows 若显示“未知发布者”或安全提醒，请确认文件来自分享者后选择继续运行。
3. 按安装向导完成安装。
4. FlowType 启动后会常驻右下角系统托盘。

## 首次设置

1. 打开 FlowType 设置页。
2. 填写自己的千问 API Key，选择千问识别服务。
3. 点击测试 API Key。
4. 选择 `Alt + Win` 或 `Win + Space` 按住说话快捷键。

API Key 使用 Windows 本机加密保存，不会包含在安装包中。分享者的 API Key 和历史记录也不会被打包。

## 使用

- 按住设置的快捷键开始说话。
- 松开快捷键停止录音并识别。
- 识别结果会粘贴到按快捷键前选中的输入框。
- 拖动绿色悬浮条可改变位置。
- 在设置页滑动尺寸滑块可调整悬浮条宽度。

## 数据位置

安装版配置和历史记录保存在当前 Windows 用户的：

`%APPDATA%\FlowType`

卸载软件不会自动分享或上传这些本地数据。

## 常见问题

- 若无法录音，请在 Windows 隐私设置中允许桌面应用使用麦克风。
- 若快捷键被其他软件占用，请在设置页切换另一组快捷键。
- 未签名安装包可能被 Windows SmartScreen 或智能应用控制阻止。正式公开分发需要使用可信代码签名证书。
